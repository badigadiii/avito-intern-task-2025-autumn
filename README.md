# Запуск
```bash
  docker-compose up -d
```

Некоторые пояснения к решению тестового задания:
- Нельзя создать пустую команду, потому что team_name обязательно должен возвращаться в users/setIsActive.
- При создании команды, если пользователь не существует, то он создаётся/обновляется существующий.
- Нельзя создать команду, если в ней есть член другой команды
